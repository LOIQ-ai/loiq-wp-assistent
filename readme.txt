== Description ==

LOIQ WordPress Agent biedt beveiligde REST API endpoints voor geautomatiseerde site debugging en beheer via Claude CLI. Inclusief read endpoints, write capabilities met safeguards, en een audit trail.

== Installation ==

1. Upload de `loiq-wp-agent` map naar `/wp-content/plugins/`
2. Activeer de plugin via het 'Plugins' menu in WordPress
3. Configureer de API key via de plugin instellingen
